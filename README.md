# dotfiles

My personal dotfiles for Arch Linux with Hyprland.

![Desktop Screenshot](.config/screenshot.png)

## Setup

This repository uses [yadm](https://yadm.io/) to manage dotfiles as a bare git repository.

To clone on a new system:
```bash
yadm clone git@github.com:mattredact/dotfiles.git
```

## Configuration

- **WM**: Hyprland
- **Bar**: Waybar
- **Launcher**: Wofi
- **Notifications**: Mako
- **Terminal**: Kitty
- **Shell**: Zsh
- **File Manager**: Yazi
- **Terminal Multiplexer**: Tmux
